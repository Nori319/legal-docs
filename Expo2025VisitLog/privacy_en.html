Privacy Policy

This App collects and handles your personal and usage data ("User Data") as described below.

1. Information Collected

  • Email address (if registered)
  • Device ID (for Firebase integration)
  • IP address and device information (for analytics and ads)
  • User-generated data such as schedules and notes

2. Purpose of Use

  • User authentication and data synchronization via Firebase
  • Usage analysis for service improvement (e.g., Google Analytics)
  • Ad delivery (e.g., Google AdMob)
  • Data sharing features with other users (only if explicitly activated)

3. Data Storage and Sharing

Data is stored on Firebase servers provided by Google LLC. Data is shared only when the user explicitly initiates sharing.

4. Third-Party Sharing

Your data may be shared with third-party services:
- Firebase (Google)
- AdMob (Google)

5. User Rights

You may request access to, correction of, or deletion of your stored data.

6. Policy Updates

This Privacy Policy may be updated as needed. Significant changes will be announced within the App.
